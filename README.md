# cursor-tutorials

A small workspace for **Cursor**-related experiments: prompts, editor workflows, agent behaviour, and anything else you want to try without touching production projects.

## What this repo is for

- Sandboxing ideas before you adopt them elsewhere  
- Short tutorials or notes you can link from other docs  
- Keeping examples version-controlled alongside your main work  

## Getting started

Clone the repository, open the folder in Cursor, and add your own tutorial folders or snippets as you go.

```bash
git clone https://github.com/TshepisoMokoena/cursor-tutorials.git
cd cursor-tutorials
```

## Contributing

This is a personal tutorial repo. Change structure or content however you like; there is no prescribed layout.
